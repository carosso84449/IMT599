Abstract:

Public uncertainty about elections has increased over the last few years, leading to distrust of election outcomes. Microsoft ElectionGuard aims to make elections end-to-end verifiable, which allows secure confirmation of votes. Our product is a website template for use by 3rd parties to host an election record with a confirmation code for each ballot. Users can use these sites to verify that their vote has been counted correctly, while keeping their actual vote confidential. This contributes to ElectionGuard’s goal of making the voting process more secure and transparent, allowing voters to have greater trust in election results.

Target Stakeholders & Users

When considering all possible stakholders and users for this project, we had to deal with various restraints. One of this restraints was the time constraint that existed due to the nature of the Practicum project timeline. Instead of having a year-long project like traditional capstone projects, we had 8 full weeks to complete the project. We also were not able to directly engage with many of the stakeholders that may end up using the dashboard (either as a voter or a developer). This meant that our approach to the project may not fully consider the needs of all stakeholders, but gives a great first example for future teams or iterations to base their work off. Implementing a uniform user interface, avoiding confusing forms, unaligned elements, and irrelevant images were key elements we wanted to include in this intial dashboard.

Summary of work & Benefits

Our product is a website template intended for use by 3rd parties to host an election record. Each election record contains cast ballots and spoiled ballots. A cast ballot can never have its contents revealed once it has been submitted. A spoiled ballot will have its contents revealed once all voting has finished. Whether cast or spoiled, each vote comes with a confirmation code. Users can use this dashboard when fully functional and operated by an independent 3rd party, to verify that their vote has been counted correctly. This contributes to ElectionGuard’s goal of making the voting process more secure and transparent, allowing voters to have greater trust in election results.
